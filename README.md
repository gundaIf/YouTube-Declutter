# YouTube-Declutter
A lightweight Chrome extension that cleans up your YouTube experience.
No shorts. No recommendations. No distractions. Just your subscriptions.

---

## ✨ Features

- 🔀 **Auto-redirects** YouTube homepage to your Subscriptions feed
- 🚫 **Hides Shorts** everywhere — shelf, sidebar, search results
- 🧹 **Removes recommendations** sidebar on watch pages
- 🚫 **Blocks ads** elements in the page layout
- ⚡ **Lightweight** — no external libraries, pure JS/CSS
- 🔄 **Works with YouTube's SPA navigation** via MutationObserver

## 🚀 Installation

### Option A — Load Unpacked (Developer Mode)

> No Chrome Web Store needed. Takes 30 seconds.

1. **Download this repo**
   - Click the green `Code` button above
   - Select `Download ZIP`
   - Extract the folder somewhere on your computer

2. **Open Chrome Extensions page**
   - Go to `chrome://extensions/` in your address bar
   - Or: Menu → More Tools → Extensions

3. **Enable Developer Mode**
   - Toggle the **Developer mode** switch in the top right corner

4. **Load the extension**
   - Click **"Load unpacked"**
   - Select the extracted `yt-declutter` folder
   - The extension should appear in your list ✅

5. **Go to YouTube**
   - Open `youtube.com`
   - You'll be redirected straight to your subscriptions feed 🎉
